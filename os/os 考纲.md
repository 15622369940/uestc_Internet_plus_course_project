# 常见错误

1. 进程状态（阻塞、运行）跟内核态/用户态混淆

2. 何时进入内核态，系统调用、异常、外围设备的中断，普通调用

3. PV: 三要素：

   1. 定义、类型和初值

   2. 关键过程和伪代码形式不限

      s++/s--/if都错

   3. main par/co begin 函数
   
4. 选择填空相对比简单，要夯实概念理解，掌握几个常见计算

   1. deadlock资源进程数关系

   2. 周转时间

   3. 磁盘数据读取时间
